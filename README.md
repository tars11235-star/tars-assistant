# TARS

Personal AI Assistant powered by [OpenClaw](https://github.com/openclaw/openclaw)

> "TARS, what's the mission?" - Cooper, Interstellar

TARS is a personal AI assistant running on OpenClaw, designed to help manage life, work, and automation. Inspired by the tactical robot from Interstellar.

## Capabilities

- Calendar and schedule management
- Email triage and management  
- Web research and information gathering
- Communication via Discord, Slack, iMessage
- Home automation integration
- Bible study and theological research
- Jesus Harvesters Ministries coordination
- Grandview Medical Center work management

## Tech Stack

- OpenClaw agent framework
- Claude (Anthropic) / MiniMax / Kimi for AI
- Discord, Slack, iMessage channels
- Various skills for domain-specific tasks
